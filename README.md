# core-schemas

All JSON Schemas used by [php-ext.com](https://php-ext.com).

## Documentation

The [docs](docs/) are generated by [adobe/jsonschema2md](https://github.com/adobe/jsonschema2md), running the following command:

```bash
jsonschema2md -d src/ -o docs/
```
